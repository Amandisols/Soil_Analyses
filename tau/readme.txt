This workspace contains the following directories:
	horizon: containing Hubbard Brook and regional site horizon data
	index: averaged values for 16 Hubbard Brook samples located deeper than 1 m in the solum, and 1 value for Brush Brook representing the nearest sample collected in the VGS deep elemental database
	exports: where png files, raw tau values, and raw elemental values are outputted
	../data/raw: raw li-borate fusion data
	extra: additional elemental data
	pitsheets: rock fragment horizon data
